# МЕТРОНОМ — Фотостудия с онлайн-бронированием

## Описание

Профессиональная фотостудия «МЕТРОНОМ» с возможностью онлайн-бронирования фотосессий. Сайт позволяет выбрать фотографа, услугу, дополнительные опции, забронировать дату и время, а также управлять своими бронированиями через личный кабинет.

---

## Возможности

### Для клиентов:
- Просмотр каталога услуг и фотографов
- Конструктор услуг (выбор фотографа + доп. опции)
- Пошаговое бронирование (4 шага)
- Проверка занятости временных слотов
- Личный кабинет с историей бронирований
- Отмена бронирований
- Добавление отзывов
- Просмотр фотографий со съёмок

### Для администраторов:
- Управление бронированиями (просмотр, статусы)
- Загрузка фотографий для клиентов
- Модерация отзывов

## Авторизация

### Регистрация:
1. Откройте `http://localhost/pages/login.php`
2. Нажмите "Зарегистрироваться"
3. Введите имя, email, пароль (от 6 символов)
4. После регистрации — автоматический вход

### Вход:
1. Введите email и пароль
2. Нажмите "Войти"

### Выход:
- Нажмите иконку профиля → "Выйти"

---

## Бронирование

### Шаги:
1. **Выберите услугу** (Портретная, Fashion, Семейная)
2. **Выберите фотографа** (Анна, Дмитрий, Елена)
3. **Добавьте доп. услуги** (визажист, стилист, и т.д.)
4. **Выберите дату и время** (проверка занятости)
5. **Подтвердите** (введите контактные данные)

### Проверки:
- Все поля обязательны
- Время не должно быть занято у выбранного фотографа
- Требуется авторизация для сохранения

---

## Личный кабинет

### Вкладки:
1. **Мои бронирования** — список с статусами
2. **Мои фотографии** — галерея с возможностью скачивания

### Статусы бронирований:
- 🟡 **Ожидает подтверждения** — можно отменить
- 🟢 **Подтверждено** — готовится к съёмке
- ⚪ **Завершено** — можно оставить отзыв
- 🔴 **Отменено** — отменено клиентом

---

## Отзывы

### Добавление отзыва:
1. Авторизуйтесь
2. Откройте `http://localhost/pages/reviews.php`
3. Выберите оценку (1-5 звёзд)
4. Напишите текст
5. (Опционально) Выберите бронирование
6. Отправьте

### Модерация:
- Отзывы со статусом `is_featured = 1` показываются в слайдере
